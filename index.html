Our Application

Our Unessay project is an educational, interactive site that aims to teach amateur web developers the importance of some of the most prevalent security risks that they must take into consideration.  We focused on Cross-Site Scripting, Broken Authentication, and Sensitive Data Exposure, but there are still many other important web application security risks out there.
